## 🛑 Остановка

Чтобы остановить Streamlit:

```powershell
# Найти процесс
Get-Process | Where-Object {$_.ProcessName -eq "streamlit"}

# Остановить по PID
Stop-Process -Id 17308

# Или остановить все процессы Streamlit
Get-Process streamlit | Stop-Process
```

## 🆘 Проблемы?

**Dashboard не открывается:**
- Проверьте, что процесс запущен: `Get-Process streamlit`
- Откройте http://localhost:8501 вручную
- Проверьте, что порт 8501 свободен

**Ошибки при загрузке данных:**
- Проверьте формат файла (CSV UTF-8 или Excel)
- Убедитесь в наличии обязательных колонок
- Попробуйте демо данные сначала

**Streamlit не найден:**
```bash
pip install streamlit
```
