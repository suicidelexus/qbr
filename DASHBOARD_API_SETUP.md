# 🔐 Настройка API через интерфейс Dashboard

## Новая возможность: Ввод API credentials прямо в Dashboard!

Теперь не нужно редактировать .env файл - все настройки можно ввести прямо в веб-интерфейсе.

---

## 🚀 Как использовать:

### Шаг 1: Откройте Dashboard

```bash
streamlit run dashboard.py
```

### Шаг 2: Откройте раздел "API Credentials"

В левой боковой панели найдите **"⚙️ Настройки API"** → **"🔑 API Credentials"**

### Шаг 3: Введите данные

**Основные поля:**
- **API URL** - URL вашей DSP платформы
  - Пример: `https://api.dsp-platform.com/v1`
- **API Key** - ключ для аутентификации (скрывается при вводе)
- **Client ID** - ID клиента (опционально)

**Кнопки:**
- **💾 Сохранить** - сохранить в сессию (действует до перезапуска)
- **🧪 Тест** - проверить подключение
- **💾 Сохранить в .env файл** - сохранить настройки постоянно

### Шаг 4: Расширенные настройки (опционально)

Разверните **"⚡ Расширенные настройки"** для тонкой настройки:

- **Timeout** - таймаут запросов (по умолчанию 60 сек)
- **Max Retries** - количество повторов при ошибках (по умолчанию 3)
- **Rate Limit Delay** - задержка между запросами (по умолчанию 0.1 сек)

### Шаг 5: Загрузите данные

Выберите **"Загрузить из API"** в разделе "📥 Загрузка данных"

**Два режима:**

1. **Последние N дней**
   - Выберите количество дней (1-90)
   - Нажмите "📥 Загрузить из API"

2. **Конкретный период**
   - Укажите даты "От" и "До"
   - Настройте dimensions и пагинацию (опционально)
   - Нажмите "📥 Загрузить из API"

## 💡 Особенности:

### ✅ Преимущества:

1. **Не нужно редактировать файлы** - все в UI
2. **Мгновенная проверка** - кнопка "🧪 Тест"
3. **Безопасность** - API Key скрывается при вводе
4. **Сохранение в сессию** - настройки действуют до перезапуска
5. **Сохранение в .env** - постоянное хранение (опционально)
6. **Гибкость** - можно менять на лету

### 🔒 Безопасность:

- API Key отображается как `••••••••` (type="password")
- Можно хранить только в session (не сохраняется на диск)
- Опциональное сохранение в .env файл
- .env файл в .gitignore (не попадет в git)

## 🎯 Примеры использования:

### Пример 1: Быстрый тест

1. Откройте Dashboard
2. Введите API URL и API Key
3. Нажмите **"🧪 Тест"**
4. Если успешно - нажмите **"💾 Сохранить"**

### Пример 2: Загрузка данных за месяц

1. Настройте credentials
2. Выберите "Загрузить из API"
3. Выберите "Последние N дней"
4. Установите 30 дней
5. Нажмите "📥 Загрузить из API"

### Пример 3: Точный период с dimensions

1. Настройте credentials
2. Выберите "Загрузить из API"
3. Выберите "Конкретный период"
4. Укажите даты
5. Раскройте "📊 Дополнительно"
6. Выберите dimensions: ['date', 'advertiser', 'campaign']
7. Нажмите "📥 Загрузить из API"

### Пример 4: Сохранение в .env для постоянного использования

1. Настройте credentials
2. Проверьте тестом
3. Нажмите **"💾 Сохранить в .env файл"**
4. Перезапустите Dashboard
5. Credentials загрузятся автоматически

## 🔧 Технические детали:

### Session State

Credentials хранятся в `st.session_state`:
```python
st.session_state['api_url']
st.session_state['api_key']
st.session_state['client_id']
st.session_state['timeout']
st.session_state['max_retries']
st.session_state['rate_limit']
```

### Создание клиента

```python
from data_layer import DSPClient

client = DSPClient(
    api_url=st.session_state.get('api_url'),
    api_key=st.session_state.get('api_key'),
    client_id=st.session_state.get('client_id'),
    timeout=st.session_state.get('timeout', 60),
    max_retries=st.session_state.get('max_retries', 3),
    rate_limit_delay=st.session_state.get('rate_limit', 0.1)
)
```

### Сохранение в .env

При нажатии "Сохранить в .env файл":
1. Читается существующий .env (если есть)
2. Обновляются значения DSP_API_URL, DSP_API_KEY, DSP_CLIENT_ID
3. Записывается обратно
4. Показывается сообщение о перезапуске

## 🆘 Troubleshooting:

### Проблема: Кнопка "Тест" не работает

**Решение:** Проверьте, что введен API Key

### Проблема: Данные не загружаются

**Решение:** 
1. Проверьте подключение кнопкой "🧪 Тест"
2. Убедитесь, что нажали "💾 Сохранить"
3. Проверьте, что за выбранный период есть данные

### Проблема: Ошибка сохранения в .env

**Решение:** Проверьте права на запись в папку проекта

### Проблема: Настройки сбрасываются при перезапуске

**Решение:** Используйте кнопку "💾 Сохранить в .env файл"
